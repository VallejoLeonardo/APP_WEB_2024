Sobre mí

Mi nombre es Leonardo Vallejo Herrera soy alumno de la Universidad Tecnológica de Durango, actualmente curso la carrera de Ingeniería en Tecnologías de la Información.

Las tecnologias de dasarrollo web que actualmente utilizamos en los proyectos que se desarrollan son: HTML para la maquetacion, CSS honas de estilo para el diseño, JavaScript para la programacion del lado del cliente, PHP para la programacion del lado del servidor y MySQL para la base de datos.
